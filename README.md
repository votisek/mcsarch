# MCSaRch

- The fastest and easiest way to start speedrunning
- Even if you are coming from Windows, or starting speedrunning as a whole, this is the easiest way to start
- This a modified Arch Linux ISO to contain automated installation steps to install pre-configured instances and tools for speedrunning

___

- A normal Arch Linux installation with archinstall, except you select a special profile from the category MCSR, in which you choose the DM/WM
- There is currently only 1 option, and that is KDE Plasma, but I am working on support for Niri
- In the future I will add more DMs and WMs
- All profiles are preinstalled with:
  - Prism Launcher (best Minecraft launcher overall)
  - Pre-configured instance with MCSR Ranked w/ optimized settings
  - Pre-configured instance with RSG mods w/ optimized settings
  - Custom script for auto setting sensitivity for boat eye
  - Pre-configured Ninjabrain bot for eye measure
  - Waywall (resize macros and mirrors) and an easy to edit configuration from (gore)
  - A sync app for creative hotbars and options.txt
  - Newest drivers for every GPU
  - A basic, but still beautiful configuration for every choice of a DM/WM
  - and even more to be coming soon
