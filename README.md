# MCSaRch

MCSaRch is a custom Arch Linux ISO focused on Minecraft speedrunning.
It provides a fast, preconfigured setup with the core tools already included.

> [!WARNING]
> This project is in early alpha.
> Some parts are still experimental and may change.

## Features

- A lightweight, Arch Linux base with preinstalled applications, such as
  - Prism Launcher
  - Waywall
  - Visual Studio Code
  - Firefox
  - Discord
  - and more!
- A selection of preconfigured\* window managers and desktop environments:
  - KDE Plasma (Recommended)
  - Jay
  - Niri
  - \**KDE Plasma is not preconfigured*
- Multiple speedrunning features so you can get started right away:
  - MCSR Ranked instance
  - Directly applied GLFW patch
  - Preconfigured Waywall with [Gore's Generic Config](https://github.com/arjuncgore/waywall_generic_config/)
- Ready to speedrun out of the box!
