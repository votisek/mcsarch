# MCSaRch

A custom Linux ISO made for speedrunning with near-perfect optimizations built on Arch Linux with an easy installation process

> [!Warning]
> This is still very early version with no real tests
> The official full release is planned for April of 2026

## Features

- Pre-configured and ready to go desktop environments
- Fast instalation and even faster gaming speeds
- Pre-installed a numera of essential programs for MCSR and more
- Pre-configured waywall and all the tools needed
- Prism launcher with RSG and Ranked added
- MPK on every instance
- MCsync set up for running with basic keybinds
