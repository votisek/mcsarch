# Keybinds

Mod is usually the windows key on your keyboard

>[!NOTE]
> Jay uses logo instead of Mod, so don't be confused in the config file.

## Workspaces

### Switch to workspace

`Mod+1` - switches to the first workspace

`Mod+2` - switches to the second workspace

`Mod+3` - switches to the third workspace

`Mod+4` - switches to the fourth workspace

`Mod+5` - switches to the fifth workspace

`Mod+6` - switches to the sixth workspace

`Mod+7` - switches to the seventh workspace

`Mod+8` - switches to the eighth workspace

`Mod+9` - switches to the ninth workspace

### Move focused window to workspace

`Mod+Shift+1` - moves the focused window to the first workspace

`Mod+Shift+2` - moves the focused window to the second workspace

`Mod+Shift+3` - moves the focused window to the third workspace

`Mod+Shift+4` - moves the focused window to the fourth workspace

`Mod+Shift+5` - moves the focused window to the fifth workspace

`Mod+Shift+6` - moves the focused window to the sixth workspace

`Mod+Shift+7` - moves the focused window to the seventh workspace

`Mod+Shift+8` - moves the focused window to the eighth workspace

`Mod+Shift+9` - moves the focused window to the ninth workspace

## Window

`Mod+Shift+F` - toggles fullscreen for the focused window

`Mod+F` - toggles floating for the focused window

`Mod+Q` - closes the focused window

## Hardware & Media

`Volume Up` - increases volume (+5%)

`Volume Down` - decreases volume (-5%)

`Mute` - toggles mute state

`Brightness Up` - increases screen brightness (+5%)

`Brightness Down` - decreases screen brightness (-5%)

`Media Play/Pause` - plays or pauses the current media

`Media Next/Prev` - skips to the next or previous media track

### Move focus across windows

`Mod+H` or `Mod+Left` - moves focus to the left

`Mod+J` or `Mod+Down` - moves focus down

`Mod+K` or `Mod+Up` - moves focus up

`Mod+L` or `Mod+Right` - moves focus to the right

`Mod+Shift+H` or `Mod+Shift+Left` - moves the focused window to the left

`Mod+Shift+J` or `Mod+Shift+Down` - moves the focused window down

`Mod+Shift+K` or `Mod+Shift+Up` - moves the focused window up

`Mod+Shift+L` or `Mod+Shift+Right` - moves the focused window to the right

## Applications

`Mod+Space` - opens a terminal

`Mod+E` - opens file manager

`Mod+D` - opens application launcher

`Mod+X` - opens window switcher

`Mod+B` - opens web browser

`Mod+P` - opens Prism Launcher

`Mod+Shift+P` - opens Power Menu

## Misc

`Mod+O` - locks the screen

`Mod+M` - logs out of the current session

`Mod+Shift+S` - Opens a screenshot UI

`Mod+R` - reloads configuration

## WM specific

### Niri

Because niri is a scrolling manager, it has the window focus and moving keybinds also with scrolling. So `Mod+Up` does the same thing as `Mod+ScrollUp` and so on.
`Mod+C` - Toggles overview mode, which shows all workspaces and windows in a grid layout

### Jay

`Mod+C` - Opens Jay control center
